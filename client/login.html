About This Project

Purpose:
•This project is a simple login page design.
Styling:
•The page utilizes the "Poppins" font from Google Fonts and has a backdrop filter for a blurred effect on the background.
Layout:
•The page uses Flexbox to center its content both horizontally and vertically, providing a responsive design..
Form Design:
•It includes fields for entering a username and password.
•Placeholder text provides guidance within the input fields.
•Icons (user and lock) are added inside the input fields for visual indication.
Button & Registration Link:
•A login button to submit the form.
•A link below the button for users to navigate to the registration page if they don't have an account.
