About This Project

Project Nature:
•Text-to-speech converter.
User Input:
•Text input via a textarea.
Voice Options:
•Users can select different voice options.
Adjustable Settings:
•Volume adjustment using a slider.
•Rate adjustment using a slider.
•Pitch adjustment using a slider.
Playback Controls:
•Play button to start speech.
•Pause button to temporarily stop speech.
•Resume button to continue speech from where paused.
•Reset button to stop speech and reset to the beginning.
User Interface:
•Designed using HTML.
•Styled with CSS.
Functionality:
•Implemented using JavaScript's SpeechSynthesis API.
Customization:
•Allows users to customize speech output according to their preferences.
Real-time Listening:
•Users can listen to the converted text in real-time.
